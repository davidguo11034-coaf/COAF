## 📖 项目简介
**COAF (AI 频率语料库)** 是一个基于 NLP（自然语言处理）技术构建的高性能词汇学习体系。我们通过深度解析海量 AI 训练级语料（如 Wikipedia & C4），结合 **多因子加权算法**，提炼出一套真正符合“真实使用频率”与“可习得性梯度”的 16 合 1 权威词汇表。

> 🌏 **学好英语全世界都是你们村，学不好英语你们村就是你的全世界。**

---

## 🛠 技术核心：数据驱动学习
不同于传统的静态词表，COAF 采用以下技术逻辑进行构建：

* **海量语料解析：** 基于 Google C4 与 Wikipedia 原始语料库。
* **多因子加权 (Multi-factor Weighting)：** 算法综合考量了 `出现频率`、`学术权重` 以及 `考试覆盖率`。
* **梯度进阶 (Gradient Progression)：** 采用由简入繁的线性路径，确保学习曲线的平滑。
